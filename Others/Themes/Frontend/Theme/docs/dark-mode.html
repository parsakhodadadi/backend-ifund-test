نسخه تیره

نسخه تیره قالب به طور پیش فرض با switcher در هر صفحه قابل فعال شدن است. اگر switcher نمی خواهید، فقط باید کد آن را از همه فایل های HTML حذف کنید.

<!-- Dark mode switch -->
<div class="modeswitch" id="darkModeSwitch">
	<div class="switch"></div>
</div>
فقط نسخه تیره

اگر تنها میخواهید از نسخه تیره قالب استفاده کنید باید فایل style-dark.css را با style.css در تمام صفحات html جایگزین کنید.

<!-- Theme CSS -->
<link rel="stylesheet" type="text/css" href="assets/css/style-dark.css">
حذف نسخه تیره

باید کد مربوط به switcher و style-dark.css را از پوشه assets/css حذف کنید.

عناوین در نسخه تیره

اگر می‌خواهید هر عنصری را در حالت تیره یا روشن مدیریت کنید، از کلاس .light-mode-item برای عناصری که می‌خواهید فقط در حالت روشن نشان داده شوند و از کلاس .dark-mode-item برای عناصری که می‌خواهید در حالت تیره نشان داده شوند استفاده کنید. به قطعه کد زیر توجه کنید :

<img class="light-mode-item" src="assets/images/logo.svg" alt="logo">
<img class="dark-mode-item" src="assets/images/logo-light.svg" alt="logo">
